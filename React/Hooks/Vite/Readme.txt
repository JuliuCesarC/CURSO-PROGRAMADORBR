Espero construir um site com uma To Do List para cada dia do mês.

A pagina devera exibir todos os dias do mês atual, ou de algum mês selecionado, cada dia devera ser um card onde deve ser possível adicionar varias tarefas.
Tentar fazer com que a organização dos cards fique parecendo com um calendário, e o dia atual fique com uma cor diferente dos demais dias.

O card terá um design de uma folha de caderno, quando não houver nenhuma tarefa no dia, devera aparecer 2 linhas vazias. O usuário poderá remover e/ou aplicar um check nesta tarefa.
Tentar fazer com que o método para inserir a tarefa seja clicando na linha vazia do card. 

Os dados devem ser salvo no LocalStorage.
